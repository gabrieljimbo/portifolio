## Dicas de uso

| Situação | O que fazer |
|----------|-------------|
| Gerar variações | Peça: *"Generate 2 more variations of this prompt"* |
| Ajustar etnia | Adicione *"Brazilian, brown skin, mixed ethnicity"* no início |
| Manter consistência de personagem | Use o mesmo prompt base, altere só um elemento por vez |
| Slider: mesmo personagem | Gere o DEPOIS → descreva o resultado → use na geração do ANTES |
| Galeria completa (4 fotos/aba) | Use Prompt A + Prompt B + 2 variações pedidas ao ChatGPT |
